# How to contribute

I'm really glad you're reading this, because we need volunteer developers to help this project come to fruition.

If you haven't already, come find us on discord! [Source Modding Pub](https://discord.gg/RZWUqkKpmC)

## Submitting changes

Please send a [GitHub Pull Request](https://github.com/SourceModdingPub/Pull-Requests/pull/new/main) with a clear list of what you've done (read more about [pull requests](http://help.github.com/pull-requests/). 
When you send a pull request, we will love you forever if you include a code sample demonstrating the expected behavior.

Thanks! :heart:
